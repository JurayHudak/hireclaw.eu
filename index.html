⚙️ Video Generation Pipeline

The system is built around a central "Project Manager" agent that orchestrates the entire workflow. It receives the initial user request, evaluates the current project state, and dynamically assigns tasks to specialized AI agents.

🎬

Creative Director

Transforms a simple user prompt into a rich, highly detailed visual concept optimized for image generation models

🔊

Sound Agent

Converts the narrative into a voiceover-ready script, ensuring correct language, natural pacing, and well-placed pauses for speech synthesis

🎥

Video Director

Designs the cinematic structure of the video, including scene composition, shot sequencing, transitions, and timing

💡 State-driven workflow — Each step (idea, prompt refinement, dialogue, assembly) is tracked in the database. The Project Manager makes context-aware decisions and can resume or adapt the pipeline at any stage.
